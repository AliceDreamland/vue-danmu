## 修改api地址
进入utils文件，修改request下的URL和adminRequest下的AdminURL。
有域名可以使用域名，没有域名IP:端口。

## 项目初始化
```
npm install
```
### 编译运行
```
npm run serve
```

### 编译打包
```
npm run build
```
